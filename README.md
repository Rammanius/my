# Сайт

Структура проекта:

- `index.html` - основная HTML-разметка страницы.
- `articles.html` - страница со статьями и практическими материалами.
- `aop-hazop-ac3-lopa.html` - отдельная продающая страница ПО для AOP/HAZOP и AC3/LOPA.
- `paz-contour-calculation.html` - отдельная продающая страница ПО для расчета контура ПАЗ.
- `css/styles.css` - все стили сайта.
- `js/main.js` - интерактивность: плавная прокрутка, активная навигация и появление карточек.
- `сайт для wix.html` - исходная версия в одном файле, оставлена как резервная копия.

Чтобы посмотреть сайт локально, откройте `index.html` в браузере.
